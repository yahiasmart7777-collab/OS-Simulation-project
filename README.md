# OS Simulation Project

This project is a simulation of an Operating System with a Graphical User Interface (GUI).  
The goal is to demonstrate how an OS manages processes, memory, files, and devices.

## Team Responsibilities
- **GUI Team**: Build the main window with 4 sections (Processes, Memory, Files, Devices).
- **Process Management Team**: Create and terminate processes, implement scheduling (Round Robin / FIFO), and show process states.
- **Memory Management Team**: Simulate memory allocation and deallocation, and visualize memory usage.
- **File System Team**: Create, delete, read, and write files, and display a directory listing.
- **Device Management Team**: Simulate devices (Printer, Keyboard) and show their status (Available/Busy).
- **Documentation Team**: Prepare the final report and video demonstration.
